# Scratch-a-Map
Project for a tool / game that lets you explore streets and places of interest near you.  
Live version: http://scratchamap.herokuapp.com/login 

## Tools used:
### Frontend:
- React (Create React App)
- Material-UI
- Axios
- JWT-Decode
- Styled-Components
- leaflet
- react-leaflet

### Backend:
- Maven
- Java 16
- Springframework
    - Security
    - Web
    - Test
    - MongoDB
- Lombok
- JUnit
- Swagger
- JsonWebToken

### Deployment:
- Docker
- Heroku
